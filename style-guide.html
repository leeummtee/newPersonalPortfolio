Interactive Elements

In-text link

CSS selector:

.link

Sample code:

<p> Click <a class="link" href="https://www.sfu.ca/"> here </a> to find out more!/ <p>

Rendered element:

Learn more here

Navigation button

CSS selector:

.button-nav

Sample code:

<a class="button-nav" href="#home"> Home</a>

Rendered element:

read more

Submit & Reset buttons

CSS selector:

.button-form

Sample code:

<input class="button-form" type="submit" value="Submit">
<input class="button-form" type="reset" value="Reset">

Rendered element

Text input field

CSS selector:

.input-text

.input-textlabel

Sample code:

<label class="input-textlabel" for="name"> Name: </label>
<input class="input-text" type="text" id = "name" name="name">

Rendered element:

Radio button input field & label

CSS selector:

input

label

Sample code:

<input type="radio" id="elem1" name="type" value="elem1">
<label for="car">element 1</label>

Rendered element:




Image as a link

CSS selector:

.block-projects

.image-link

Sample code:

<div class = "block-projects">
<a href = "https://www.sfu.ca/">
<img class="image-link" src="compressed-images/placeholder.jpg" alt="trim cleanse">
</a>
</div>
